# RESTful API for Article Management

Node.js and Express REST API for article CRUD operations with image upload support.

## Overview

This project manages articles through a MongoDB-backed model layer and exposes routes for creating, reading, updating, deleting, and uploading media associated with articles.

## Stack

- Node.js
- Express
- MongoDB with Mongoose
- Multer for file uploads
- Validator for request validation

## Scripts

```bash
npm install
npm start
```
